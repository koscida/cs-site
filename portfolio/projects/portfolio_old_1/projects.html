Magic Mailbox


This was a project I worked on during my frist year in a freshmen projects class (GEEN 1400). We built an oversized mailbox that utilized a touchscreen tablet as an interface for kids ages 4-7 to email and send pictures with. My job was creating the GUI for the kids to use, as well as testing. Simplicity and functionality drove my design for the interface, which was especially since I was creating a product primarly for an audience with limited reading capibilities. I built the project using Labview, which was a challenging yet fun because I had to teach myself everything and learned to work around the limited capabilities of the language. The project was submitted at the design expo and won runner-up for User's Choice Award in Spring 2009.
