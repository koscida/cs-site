XBOX Game

This is a project for a class I'm taking currently, called XBOX: Advanced Game Development. We are working in a group of 9, and my position involves interface design and GUI development. In the class, we have the entire semester to create an XBOX game, the game design is currently in progress and we are in the beginning stages of creating a cohesive idea.

The end deliverable for the class will be a demo of the greater idea, either the first level or a later level that will be used to hook the players. We plan on continuing this project into the next semester, hopefully to create another level of the same game.
