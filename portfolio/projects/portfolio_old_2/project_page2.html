Hotspots

This was a team project I did for the ATLS 3020: Digital Media 2 class. It is written in PHP and uses a MySQL database. I worked in a team of 3 where I was the primary software developer.

Hotspots is an online roadtrip planner and vacation resource. My team and I researched different 'Hotspots' or vacation spots around the Western US and populated a database with the information. We built the website around this data and allowed users to create accounts and comment on the hotspots. We then build a roadtrip planner that had the capibility to add different hotsopts and map out the trip.
